# React Application Template with TypeScript From Scratch

## Dependencies

Node.js version

```sh
$ node -v
v14.16.0
```

## Start Project

```sh
$ yarn
$ yarn start
```

## Build for Production

```sh
$ yarn build
```

## Run Linter and Prettier

```sh
$ yarn lint
$ yarn format
```
